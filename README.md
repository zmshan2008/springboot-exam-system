# 学生自助考试系统
后端技术栈： springboot+springmvc+mysql+mybatis+redis
# 前台： 
**考试功能**：选择考试，答题，客观题自动评分；  
**论坛功能**：发帖，回帖，回复；  
**用户功能**：修改密码，修改个人信息，查看参与过的考试，发帖记录；  
# 后台：
对支撑整站需要用到的数据进行增删改查。对答案主观题打分，管理考试，管理题库，管理帖子等等；
